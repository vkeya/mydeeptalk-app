# MyDeepTalk Product Roadmap

This roadmap outlines the long-term development journey for MyDeepTalk.

# Stage 1 — Foundation Platform (Current)

## Goal

Build a trusted platform connecting emotional wellness tools with professional support.

## Features

### Authentication

* Email/password signup
* Google login
* Email verification
* Terms and privacy acceptance
* Privacy aliases

### Therapist Marketplace

* Therapist profiles
* Credentials verification
* Professional services agreement
* Admin approval workflow

### Client Journey

* Dashboard
* Pre-booking intake
* Therapist matching
* Session booking

### Payments

* IntaSend integration
* Gift sessions
* Healing circles

### Sessions

* Google Meet integration
* Session reminders
* Booking management

### Reflection

* Journaling
* Self-assessment

### Community Support

* Reviews
* Dashboard announcements
