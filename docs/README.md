# MyDeepTalk

MyDeepTalk is an emotional wellness, self-discovery, and therapist support platform built to help people understand themselves, reflect privately, and connect with verified therapists when they need deeper support.

## Mission

To make emotional wellness more accessible, private, preventive, and culturally relevant, starting from Africa and expanding globally.

## Core Purpose

MyDeepTalk helps users:

- Reflect through journaling
- Complete self-discovery check-ins
- Prepare before therapy
- Find verified therapists
- Book private therapy sessions
- Receive support through gifted sessions and healing circles

## Tech Stack

- Next.js
- TypeScript
- Firebase Authentication
- Firestore
- Firebase Storage
- Cloudinary
- IntaSend Payments
- Google Calendar
- Google Meet
- Vercel

## User Roles

### Client

A person seeking emotional wellness support, journaling tools, self-assessment, therapist matching, and therapy sessions.

### Therapist

A verified professional who creates a profile, uploads credentials, signs the professional services agreement, sets availability, and manages bookings.

### Admin

A platform manager who reviews therapist profiles, approves therapists, and monitors platform quality.

## Main Features

- Client signup and email verification
- Therapist signup and professional agreement acceptance
- Therapist profile and credential upload
- Admin therapist approval
- Pre-booking intake form
- Therapist discovery and filtering
- Session booking
- IntaSend payment flow
- Google Meet session creation
- Email notifications
- Journaling
- Self-assessment
- Gift therapy
- Healing circles
- Client privacy alias system
- Therapist reviews
- Dashboard awareness announcements

## Privacy Approach

Clients can choose a privacy name or alias. This alias is shown to therapists and in reviews instead of exposing the client’s real name.

## Current Deployment

The app is deployed on Vercel and uses Firebase for authentication and database services.
